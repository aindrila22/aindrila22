### 🚀 What I'm building

**[MockExperts](https://mockexperts.com)** — Unified AI technical hiring infrastructure, built entirely solo.

> 200+ users · 200+ sessions · 9 countries · 69.2% completion rate

- Engineered a **6-agent orchestration pipeline**: Orchestrator → Q-Agent → Eval-Agent → Interviewer → Anti-Cheat → Hiring Committee
- Built a **3-agent Hiring Committee** (Tech Lead + Bar Raiser + Hiring Manager) with context compression — 90% token reduction via Performance Log + Smart Transcript Slicing
- Implemented **dynamic model routing**: `grok-4.3` / `grok-4-1-fast-non-reasoning` with GPT-4o fallback across 3 task tiers
- Real-time **WebSocket interview loop** with proctoring (tab-switch detection, face-loss tracking, copy-paste interception)
- **B2B assessment engine** with custom `evaluationRubric` — per-company weighted scoring across DSA, System Design, MCQ, Communication, Behavioral
- Dual payment gateways: **Paddle** (USD) + **Razorpay** (INR)
- Stack: `Node.js` · `Express` · `MongoDB` · `Redis` · `Next.js` · `WebSockets` · `xAI Grok` · `OpenAI`
